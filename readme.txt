WPLMS GA is an integration of Google Analytics with WPLMS

== Description ==
WPLMS GA plugin is developed for integrating WPLMS theme with Google Analytics. This plugin allows you to track clicks on take this course button of each course. It also allows you to track the unit or quiz the student is currently on, in the course status page which loads using ajax. It also allows you to track the questions on which the student is on, in the single quiz page (questions loaded using ajax).

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'WPLMS ga'
3. Activate WPLMS GA from your Plugins page. 

= From WordPress.org =

1. Download WPLMS GA.
2. Upload the 'wplms_ga' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate WPLMS GA from your Plugins page.

= Once Activated =

1. Visit 'LMS -> Settings -> General -> Add GA Tracking ID' and add your google analytics tracking ID here.
